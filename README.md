# cpp5homework
# 언리얼 로그를 이용한 프로그램

## 기능

### 좌표 이동

언리얼 에디터로 플레이 시 랜덤으로 10번 좌표를 이동하여 `좌표`와 `이동 거리`를 로그에 출력합니다.

### 이벤트 호출

10번 이동하는 동안, 50 퍼센트의 확률로 이벤트를 호출합니다.

### 총 결과 반환

10번의 이동을 마친 후, 총 이동 거리와 이벤트가 발생한 총 횟수를 출력합니다.
